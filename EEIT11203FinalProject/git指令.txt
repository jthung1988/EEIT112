命令:
git clone --第一次把檔案從遠端(Github)抓下來使用
git pull --從遠端更新git本地資料夾，意思是第二次以後不用clone，用pull更新自己的檔案
git status --查看檔案更新狀況
git add . --把檔案加入git檔案更新追蹤
git commit -m "Message" --把檔案跟新至git本地資料夾，並記錄版本註記 Message
git push --把檔案更新至遠端資料夾(Github)
git init --把這個資料夾設為git本地資料夾
